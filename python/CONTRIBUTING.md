# Python Component

## Contributing

Start by installing the package:

```bash
pip install poetry
poetry install
```

You can then run commands locally as follows:

```bash
poetry run hal9 create my-project
```

Globally install hal9 as follows:

```bash
pip install .
```

### PyPi

Alternatevely,

```bash
pip install hal9
```

Followed by running as

```bash
hal9 create my-project
```
