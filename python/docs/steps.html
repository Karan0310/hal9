class hal9.hal9#

hal9 pipeline class

add_step(step_name, **kwargs)#

Adds a generic step to a pipeline instance.

This is

>>> pipeline = h9.create()
>>> pipeline.add_step("load")
>>> pipeline.show()
load(**kwargs)#

Add a load step to the pipeline.

This function wraps the add_step method and return the pipeline with a load step added.

>>> pipeline = h9.create()
>>> pipeline.load(pandas.DataFrame({"col1": [1,2], "col2": [1,2]}))
>>> pipeline.show()
show(height=400, **kwargs)#

Renders the pipeline content on a notebook.

This functions produces a HTML display block rendering the pipeline content.

>>> pipeline = h9.create()
>>> pipeline.add_step("load")
>>> pipeline.show()
