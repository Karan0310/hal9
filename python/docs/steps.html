Our first chart type needs no introduction

  • y (str) – The variable that should be on the vertical axis.

  • orientation (str) –

  • palette (str) – Colors to use for the different levels of the y variable. Should be one of the valid d3.js color palettes.

  • fontsize (str) – Size of the font to be used in the x and y axes.

  • tickrotation (str) – The angle at which to place the x-axis labels

  • marginleft (str) – the left margin

  • marginbottom (str) – the bottom margin

facets_chart(x, y, facets, color, chartType, palette)#

A chart of charts organized in a grid

Parameters

  • x (str) – The column in the dataframe that should be along the x-axis in all the subplots.

  • y (str) – The column in the dataframe that should be along the y-axis in all the subplots.

  • facets (str) – The columns in the dataframe that should define the marks to facetted subplots in horizontal and vertical directions of the subplots.

  • color (str) – The column in the dataframe that will define the color of the marks in each of the subplots.

  • chartType (str) – The type of chart each of the subplots. Currently supports Scatter, Bar, Line and Cell

  • palette (str) – Colors to use for the different levels of the y variable. Should be one of the valid d3.js color palettes.
